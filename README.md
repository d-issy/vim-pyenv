# vim-pyenv
this plugin is automatic sys.path changer for pyenv

use it with jedi-vim

## Requirement
- pyenv
- python3
- vim --with-python3

## Install with Plug
```vim
Plug 'davidhalter/jedi-vim', {'for': 'python'}
Plug 'shepabashi/vim-pyenv', {'for': 'python'}
```
